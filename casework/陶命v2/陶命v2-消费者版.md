**你的财运特征**：日坐正财 = 靠本事赚钱的根基在，但身弱不胜财 = 赚一块钱要花别人三倍精力。午戌半合火 = 赚的钱容易被"合走"（人情/意外/冲动消费）。

| 时间段 | 策略 | 人话 |
|---|---|---|
| **当下** | 工资到账当天自动转30%到定期 | 活期的钱会被"合走" → 只有碰不到的钱才是你的钱 |
| **当下** | 不追大财 · 赚钱节奏=工资+小幅副业 | 身弱扛不住大财=贪多嚼不烂 |
| **47岁后** | 转型"脑力变现" → 教学/咨询/写作 | 食伤生财=终于可以"轻松赚"了 |
| **终身红线** | ❌ 不炒高风险品种 ❌ 不给人担保 | 正财命走偏财路=翻车概率极高 |

### 4.3 健康 ⭐⭐ ｜ "最大弱区 → 省电模式生存"

**你的身体特征**：壬水身弱（只有19%的能量）要扛住土38%+火15%的消耗 = 别人熬一晚恢复3天，你要恢复一周。你的身体是"省电模式"——不是你体质差，是出厂设置就是这样。

**三大高风险方向**：
1. 🔴 **肾和泌尿**（土克水）→ 每季度查肾功能+尿常规
2. 🔴 **心血管/失眠/血压**（火旺耗水）→ 2026年双重丙午=今年最大健康风险
3. 🟡 **情绪内耗/焦虑**（戌戌自刑）→ 自我折磨比外部压力杀伤力更大

**每日必做清单**：
| 优先级 | 事项 | 原因 |
|---|---|---|
| 🔴 P0 | 11点前入睡 · 不可谈判 | 壬水身弱最怕熬夜=透支最稀缺的资源 |
| 🔴 P0 | 每周游泳2次 | 水行运动=给壬水充电 · 效果是其他运动的2倍 |
| 🟡 P1 | 每天5分钟日记 | 打断戌戌自刑循环=降低情绪内耗 |
| 🟡 P1 | 每季度体检（肾+心+血压） | 土克水+火耗水=重点监控区 |

**2026年特别警示**：
- 🔴 **10月（戊戌月）= 全年最危险月份**——三戌聚首 · 什么都不做只做一件事：体检+休息
- 🟡 **全年**：双重丙午火旺 → 心血管/血压/头疼/失眠是今年主旋律 → 感到不舒服**立刻去医院**不要扛

### 4.4 婚姻 ⭐⭐⭐ ｜ "不简单但有救 → 维护婚姻=维护财运"

**你的婚姻特征**：官杀混杂 = 感情线天生比别人复杂。日坐正财+正官 = 你的钱和婚姻绑在一起——好婚姻=好财运，差婚姻=漏财。
