## Part 1 · 命局核心结构

### 1.1 日主旺衰判定 ｜ 权重 100/100 ｜ 置信 ⭐⭐⭐⭐⭐

> **日主**：辛金（阴金）· 生于戌月（霜降后）

**【第一步：得令判断】**
- 辛金生于戌月 → 戌月土旺金相 → 金处"相"地（不当令但有余气）
- 结论：**半得令**（不如酉月申月当令，但戌中藏辛金本气 = 仍有根气）

**【第二步：得地判断】**
- 年支戌中藏辛（中气） → ✅ 有根
- 月支戌中藏辛（中气） → ✅ 有根（双戌双根）
- 日支巳中藏庚（中气） → ✅ 有根（庚为劫财，同类助力）
- 时支酉为辛金禄地（本气辛） → ✅ **强根**（时支坐禄 = 晚年有靠）
- 结论：**得地极强**（四柱地支处处有金根，尤其时支酉金坐禄）

**【第三步：得生判断】**
- 年支戌中戊土（本气）生辛金 → ✅ 有生
- 月支戌中戊土（本气 · 月令加权）生辛金 → ✅ **强生**
- 日支巳中戊土（余气）生辛金 → ✅ 有生（弱）
- 结论：**得生极强**（双戌印库重重 = 印星生身力量过剩）

**【第四步：受克判断】**
- 日支巳中丙火（本气）克辛金 → ⚠️ 正官克身（但被巳酉半合金局减弱）
- 时干丁火克辛金 → ⚠️ 七杀克身（但丁壬合木 = 七杀被合绊，克力大减）
- 年干壬水泄辛金 → ⚠️ 伤官泄身（但壬水独透，泄力有限）
- 结论：**受克受泄均弱**（官杀虽有，但被合局 / 合绊化解大半）

**【综合判断】**
```
半得令(+1) + 得地极强(+3) + 得生极强(+3) + 受克弱(-1)
→ 日主 **身旺偏强**（非从旺，因有官杀克制 + 伤官泄秀）
→ 置信度：⭐⭐⭐⭐⭐
```
