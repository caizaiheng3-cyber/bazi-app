### 维度 ① · 强弱 vs 喜忌反差（D5）— 一生主旋律

> 此维定调："命主一生的力学方向是顺还是逆，对抗强度多大。" 这是所有其他维度的底色。

#### 1.1 🔴 主线级 · 忌神水过旺至 53%（远超均衡值 20%）

`[贴身度: ★★★★★ 全盘四柱透干]` `[宫位: 贯穿四柱-一生持续]` `[触发: 常态发力]` `[清浊: 浊]` `[调候: 致命]` **影响因子: 95/100 [外显]**

**📜 命理事实**
癸水透年、月、时三干 + 壬水居日干 + 月支亥（水本气）当令 + 年支酉金生水 + 日支戌中辛金生水 + 时支卯虽伤官泄水但不敌——**水势汪洋成专旺之象**。占比 53% 远超平均 20%，是局中绝对主力。

**🎯 对命主的影响 + 趋利避害**
- **影响**：一生**所有事都被"水"的属性裹挟**——智力溢出、思绪过载、想得多做得少；情感濡湿、情绪反复、夜不能寐；交际多、人脉广但留不住核心朋友；肾/泌尿/血液系统天生压力大。
- **核心矛盾**：不是"少了什么"而是**"多到溢出无处安放"**——水太多必然找出口，找不到就反噬。
- **趋利**：① 顺势走"用水"行业（流通、信息、贸易、互联网、流量、内容、咨询、心理、水产、酒水、海运）—— 让水"有事干"；② 多人交往中做"信息节点"而非"决策中心"，让水势变现；③ 居住/工作选靠水的城市/楼层（沿海、近河、高层）反而稳定。
- **避害**：① **绝对禁止**走"克水"路径（强行立规矩、做严格管控、对抗权威）—— 水反弹必伤己；② 忌长期独处—— 水气无人消耗会内化成抑郁/焦虑；③ 忌深夜熬夜—— 水旺人本就肾压大，熬夜直接击穿底线。

#### 1.2 🔴 主线级 · 主用神火仅占 6%（致命级调候缺位）

`[贴身度: ★ 仅藏日支戌中余气]` `[宫位: 日柱-壮年才显]` `[触发: 等大运透火才发力]` `[清浊: 浊-被合走]` `[调候: 致命]` **影响因子: 92/100 [实质]**

**📜 命理事实**
亥月（冬）大雪后 0 天，**水势初旺木气将生**——按《穷通宝鉴》，亥月调候**首取丙火暖局，次取戊土制水**。蔡命局中：丙丁火**全无透干**，仅戌中藏丁（中气）—— 6% 的火气还被戌卯六合化掉一部分。**调候致命缺位**。

**🎯 对命主的影响 + 趋利避害**
- **影响**：① **身体打底层就缺阳气**——畏寒、循环差、消化弱、情绪易低落；② **决策缺"明断"**——什么都看得见但下不了狠手，因为"火"是决断之力；③ **人生缺"温度感"**——做事容易冷处理、不擅长煽情/带动情绪/激发团队。
- **核心矛盾**：火不是"少了一点"，是**几乎没有**—— 必须靠**外部火**来续命，不是"内修"能补的。
- **趋利**：① 主动靠近"火属性"的人/事—— 阳光型伴侣、热情同事、做演讲/视频/直播这种"靠火热情接管人"的工作；② 居住/装修偏暖色调，南向房间，多晒太阳（这是物理意义的补火）；③ 35-50 岁壮年是用神火的"主导期"，要在这一段主动制造"温度场景"——结婚、生子、做有热度的事业；④ **运动选有氧（心肺=火）**，跑步、骑行（注意是用火带水，不是泡冷水）。
- **避害**：① 忌做需要"冷酷决断"的工作（外科、刑侦、纪委）—— 你扛不住；② 忌长期单身—— 没有伴侣的火，自身的水会无止境扩张；③ 忌养水族/在阴湿环境工作—— 等于"火上浇水"。

#### 1.3 🔴 主线级 · 主用神土仅占 9% 且被合走（结构性缺位）

`[贴身度: ★★★★ 藏日支但被时支合]` `[宫位: 日柱-夫妻宫被夺]` `[触发: 戌卯合常态生效，被合走是常态]` `[清浊: 极浊]` `[调候: 严重缺位]` **影响因子: 90/100 [实质]**

**📜 命理事实**
戊土仅藏日支戌中（本气），9% 的份额本就薄；偏偏**戌+卯六合化火**——戌中戊土被卯（伤官木）合化为火，**用神土被时柱直接夺走**。这是蔡命**最致命的结构性缺陷**。

**🎯 对命主的影响 + 趋利避害**
- **影响**：① **"自律 / 边界感 / 制度感"先天薄弱**——土代表克水的能力、自我管理、规则意识，土被夺 = 一生在"想自律但自律不起来"中循环；② **七杀（戊）被夺等于"内在权威"消失**——容易抗拒被人管，但又渴望被管；③ **配偶宫被合**—— 婚姻里"对方想给你立规矩"和"你又抗拒"反复拉扯；④ **脾胃 / 皮肤 / 肌肉**长期偏弱。
- **核心矛盾**：土不仅是"少"，是**有但被夺**—— 你能感觉到"我本该有这个能力"，但伸手就抓不住。
- **趋利**：① **必须建立外部约束系统**——教练 / 私教 / 健身 App 打卡 / 项目制工作 / 强 KPI 团队 / 写日报对外公开 —— 用外部规则补内部缺失；② 选**配偶时倾向于"温和而坚定"型**，不要选纯粹强势型（合不化你）也不要选纯弱势型（撑不住你的水）；③ 做**实业 / 重资产 / 不动产 / 农业 / 建筑 / 矿产** 的边缘工作（哪怕只是投资）—— 借土的"沉重感"压住自身的水。
- **避害**：① 忌纯靠"自我意志"立规矩—— 一个月内必崩；② 忌在卯（兔）年、未（羊）年签长期合约 —— 卯未会拱木局再夺戌土，那两年立的规矩 90% 会失效；③ 忌做"灵活自由职业"无任何外部 deadline 的工作—— 你的水会把所有时间冲散。

#### 1.4 🟠 重要级 · 日主与喜忌方向相反（极旺顺势 vs 内心抗拒主流）

`[贴身度: ★★★★ 日主自身]` `[触发: 一生持续]` **影响因子: 75/100 [外显]**

**📜 命理事实**
日主壬水极旺（水占 53%），按"极旺从势"的取用原则，应当**顺水之势取金水为助**——但实际局中已经金水超载，再加只会更失衡，所以**真正的解法是"用泄不用克"**：用木（伤官、食神）去**疏通**水气，而非用土去**堵截**。然而命主主观感受上往往是"想立规矩、想做正事"，**内心方向 vs 命运方向相反**。

**🎯 对命主的影响 + 趋利避害**
- **影响**：① **道理上知道该顺应**，但情感上**总想反抗主流**——越压越想冒头；② 一生在"想做严肃正经事"和"实际靠灵活的水生意挣到钱"之间反复横跳；③ 容易高估"对抗"的价值，低估"借势"的力量。
- **趋利**：① 接受**"我就是吃水的饭"**—— 别强行去做"火土型工作"（管理、行政、传统制造）；② 把"反抗主流"的冲动**转化为"差异化定位"**—— 在水属性赛道里做小众、做创新、做反共识，但不要跑去敌对赛道；③ 30-35 岁是关键转折，必须在那时认清自己的"水命属性"，否则会越走越拧巴。
- **避害**：① 忌做**纯执行类、纯管理类**工作—— 你的水撑不起；② 忌**为了证明"我也能正经"** 而入错行—— 这种自证心态会浪费 5-10 年。

**🎯 对命主的影响 + 趋利避害**

`[贴身度: ★★ 仅日支]` `[宫位: 日柱-壮年才能用上]` `[触发: 等大运透出]` `[清浊: 浊]` `[调候: 致命]` **影响因子: 88/100 [实质]**

- **避害**：① 忌**长期在阴湿环境**（地下室、潮湿仓库、海边一楼）—— 等于天天加固冰冻；② 忌**纯素食 / 节食 / 长期断食** —— 你的火本来就不够，能量来源还断绝；③ 忌**冬季泡冷水 / 冰浴 / 极寒训练** —— 短期刺激爽，长期压垮你的火。

### 一生主线（按权重 / 时序）

#### 主线一：**接受"水命"，把水做到极致**（贯穿一生）
你的水**真材实料**（4 位水全通强根），不是装的也藏不住。一切对抗水的努力都是损耗，一切顺势用水的努力都会被命运加倍奖励。**职业、城市、伴侣、爱好——全部围绕"水"选**。

#### 主线二：**用木（输出 / 创作 / 表达）疏通水势**（17 岁起持续）
水→木 是命中第一断点，**唯一解法是强行建立"输出节奏"**。少一天写作 = 多一天内耗。

- 49+ 岁：回归创作，靠作品/教学立身

#### 主线三：**用外部制度补"内在七杀缺位"**（终身）
七杀被合走 = 你的"内在权威"先天缺席。**靠自我意志立规矩永远会崩**——必须建立 **"教练 + 制度 + 强配偶"** 的三层外部约束。

#### 主线四：**用物理性补火**（每年 11 月 + 35-50 岁壮年期）
调候致命缺位**靠内修补不了**——必须用居住地、运动、阳光、伴侣、饮食的物理方式补。**南方城市 + 早起晒太阳 + 温补饮食 + 阳光型伴侣**是命主的"续命四件套"。
