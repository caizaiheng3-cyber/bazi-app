## 模板设计说明

系统对同一份八字数据提供两种输出视角，通过界面上的 **视角切换** 按钮自由切换：
- **专业模式**（服务提供者视角）：面向命理师/运营者，强调完整性、可追溯性
- **用户模式**（消费者视角）：面向终端用户，强调易读性、实用性

## 📋 日主强弱判断（推断过程）

> ⚠️ 以下为逐步推断过程，用于校验结论是否准确

```
【第一步：得令判断】
  日主丙火，生于申月（金旺之月）
  申月火处"囚"地 → ❌ 失令
  结论：日主失令

【第二步：得地判断】
  寅中藏甲木（生火）、丙火（本气）→ ✅ 日支有根
  午中藏丁火（帮身）              → ✅ 年支有根
  结论：日主得地（有两处根气）

【第三步：得生判断】
  甲木（月干）生丙火              → ✅ 有生
  乙木（时干）生丙火              → ✅ 有生
  结论：日主得生（两干生扶）

【第四步：受克判断】
  庚金（年干）克甲木（印星受制）   → ⚠️ 印星受克
  申中庚金克寅中甲木               → ⚠️ 寅申冲
  结论：生扶力量受到一定制约

【综合判断】
  失令(-2) + 得地(+2) + 得生(+2) + 印星受克(-1)
  → 日主偏旺（中等偏上）
  → 置信度：⭐⭐⭐⭐ 高

【用神推断】
  日主偏旺 → 宜泄宜耗宜克
  → 用神：土（食伤泄秀）、金（财星耗身）
  → 忌神：木（印星生身）、火（比劫帮身）
  → 闲神：水（官杀克身，但力量较弱）
```

# 模板二：消费者视角（用户模式）

> **设计原则**：
> 1. 报告结构：**共情 → 解释 → 出路**
> 2. 命理特质五要素：**命格意象 · 时间节奏 · 术语点缀 · 天命感 · 具体指引**
> 3. 区别于心理分析：归因于命格（非个人缺陷）、指向未来（非回溯过去）、适度神秘感

### 🏢 事业发展（您关心的重点）

#### 「共情层」—— 先读懂你的处境

您可能经常有这样的感受：明明自己能力不差，但总觉得机会来得比别人慢；有时候会觉得自己做了很多，但回报没有达到预期。

#### 「解释层」—— 你的事业命盘解读

- **你的事业DNA**

  你不是那种安于现状的人。偏财格的人天生就对"可能性"有嗅觉——比起铁饭碗，你更适合有空间施展拳脚的舞台。管理、营销、投资、自主创业，或者与文化、教育、媒体相关的领域，都是你命格中写好的方向。

**👥 人际关系**

# 两种模板的对比总结

| 维度 | 服务提供者模式 | 消费者模式 |
|------|-------------|-----------|
| **信息量** | 100%（完整原始数据） | 约 30%（精选关键结论） |
| **专业术语** | 大量使用，不做简化 | 约 10% 自然融入，出现时附通俗解释 |
| **推断过程** | 完整展示每一步逻辑 | 隐藏过程，仅展示结论 |
| **专业依据** | 每个结论都有规则出处 | 约 10% 的结论附带简短命理依据 |
| **语言风格** | 学术/技术报告风格 | 温暖 + 命理意象感（"你的命属炉中火"） |
| **报告结构** | 数据维度分块展示 | 共情 → 解释 → 出路 三层叙事 |
| **交互方式** | 无，直接展示全量数据 | 先选关心领域，再定制输出 |
| **归因方式** | 客观数据分析 | 归因于命格（"不是你不好，是命格使然"） |
| **时间维度** | 大运流年数据列表 | 确定性的"时间地图"（"2028年是黄金窗口"） |
| **置信度标注** | 有（高/中/低） | 无（避免用户困惑） |
| **负面信息** | 如实标注（凶煞、冲克） | 天命感语言转化（"命格提醒你…"） |
| **开运指引** | 无（纯数据） | 幸运颜色/方位/数字/饮食等命理专属建议 |
| **可导出** | 支持原始数据导出 | 支持图片导出（适合分享） |

## 消费者报告中命理特质五要素的分布

| 要素 | 在报告中的位置 | 示例 |
|------|-------------|------|
| **命格意象** | 开篇（命理人设锚点） | "你的命属炉中火…外表沉稳，内心有一团不灭的火焰" |
| **天命感** | 共情层 + 其他概览 | "不是你不够努力，而是你的命格注定了你是厚积薄发型的人" |
| **术语点缀** | 解释层的 📎 依据标注 | "偏财格配文昌，利商业策划与文化类事业" |
| **时间节奏** | 出路层（时间表） | "2028-2037年是你事业的黄金窗口" |
| **具体指引** | 贵人画像 + 开运指南 | "贵人是年长沉稳的女性""多穿黄色""多食山药" |

- 匹配逻辑：场景 + 关键词 → 命中预设的 `shifuReplies` 中的某条 → 返回完整 `ShifuReply`
- 命不中时：返回兜底回话「先生需细思，请稍候……」+ 一条通用三段式回话
- 这是 Mock 阶段的"伪 AI"，M7.4 真实引擎做完后替换
