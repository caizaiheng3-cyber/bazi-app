### 🚨 P7.2 健康脏腑评分修复（含字段名 Bug）

3. 在 ratio 判定基础上叠加用忌神修正：

### ⚠️ P7.4 官非评级可解释性修复

**问题定位**：`legalRiskAnalyzer.ts → 主入口 summary 拼接`
- 原 summary：`综合官非风险等级：【中等关注】（★3/5）。原局未带凶神组合。首个高危年：2012年...`
- 消费者读完困惑：**"既然原局没凶神，为啥还是中等关注？"**
- 真实原因：评级算法考虑了 `highEvents`，但 summary 没说

**修复方案**：
summary 改为三层显式拆解：
1. **【原局基础】**：明确说"命格本身不带 X/Y/Z 凶神"或"命格带 N 项静态风险"
2. **【流年触发】**：明确说"一生 N 个潜在官非年（高危 X 个 / 中等 Y 个）"
3. **【评级依据】**：根据"原局 × 流年"的四象限给出可解释结论：
   - 原局清净 + 流年无高危 → 真正低风险
   - 原局清净 + 流年高危 → "潜在被动风险，重点在防"
   - 原局有凶 + 流年清净 → "停留在性格层面，修身可化"
   - 双层叠加 → 高度警觉

**蔡命预期效果**：
> 综合官非风险等级：【中等关注】（★3/5）
>
> **【原局基础】**：命格本身不带羊刃/三刑/伤官见官等凶神组合，**先天无重大法律隐患**
>
> **【流年触发】**：一生 6 个潜在官非年（高危 4 个 / 中等 2 个）
>
> **【评级依据】**：虽然原局清净，但一生有 4 个流年高危事件被触发（如羊刃逢冲、伤官见官、日柱反吟），**评级反映的是"潜在被动风险"，不是"主动作恶倾向"**——重点在防而非治
