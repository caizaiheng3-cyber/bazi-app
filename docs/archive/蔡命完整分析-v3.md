### 1.1 日主旺衰判定 ｜ 权重 100/100 ｜ 置信 ⭐⭐⭐⭐⭐

| 推断步骤 | 判定 | 依据 |

## Part 5 · 消费者版精华

> **设计理念**：消费者版不是"专业版的简化"，而是"专业版的转译"——把命理黑话翻译成普通人能懂能用的语言，并且**每一句结论都要带"对你来说意味着什么"+"该怎么做"**。

### 🎴 第一张牌 · 你的命运底色

> **江河汪洋之命**：你是大江大河，不是小溪小池。
> **力量超标**——主见大、能量大、想得多；你比身边 90% 的人都"想得明白"，但也比他们更"做得拧巴"。
> **核心矛盾**：你天生力量太多，必须找出口；找到出口（创作 / 表达 / 流量 / 信息行业）就一飞冲天，找不到就内耗成抑郁焦虑。

### 🎴 第二张牌 · 你的开运钥匙

> **方向 = 南方**（深圳 / 广州 / 厦门 / 海口比北方好太多）
> **颜色 = 红橙黄**（衣服 / 内饰 / 配饰常备暖色）
> **行业 = 火土**（能源 / 教育 / 影视 / 餐饮 / 房产 / 化工）
> **时间窗口 = 41 岁后**（你的人生上半场是"打底"，下半场才是"开张"）
> **配偶 = 火属性人**（生肖虎马狗 / 农历 4-7 月生 / 阳光独立型）

### 🎴 第三张牌 · 你这辈子的三个"千万别"

> 1. **千万别合伙做生意**——4 个比劫围攻 1 个财星，合伙必败；要做就独资。
> 2. **千万别碰投机**（股票 / 期货 / 币 / 赌博）——偏财缺位，快钱不归你，10 万投进去能拿回 1 万都是好的。
> 3. **千万别熬夜**——水旺人本就肾压大，熬夜直接击穿身体底线，30 岁后报应翻倍。

### 🎴 第四张牌 · 你这辈子的三个"必须做"

> 1. **必须把"表达 / 创作"产品化**——写书 / 做课 / 做 IP / 开咨询，这是你最大的财富杠杆。
> 2. **必须读到硕士及以上**——印星 + 官印相生 = 学历对你是乘数效应，不读太亏。
> 3. **必须 33 岁后再结婚**——早婚必离，晚婚才稳；2026 / 2037 是命中两个最佳窗口。

### 🎴 第五张牌 · 你的人生时间表
