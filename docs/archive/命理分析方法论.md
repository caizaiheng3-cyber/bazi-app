# 第一章：基础知识体系

## 1.1 阴阳

阴阳是命理学的最底层哲学框架。万物分阴阳，天干地支亦分阴阳。

| 属性 | 阳 | 阴 |
|------|----|----|
| 天干 | 甲、丙、戊、庚、壬 | 乙、丁、己、辛、癸 |
| 地支 | 子、寅、辰、午、申、戌 | 丑、卯、巳、未、酉、亥 |
| 特质 | 外向、主动、刚健、进取 | 内敛、被动、柔顺、收藏 |

**在分析中的应用**：
- 阴阳属性决定十神的"正"与"偏"（同性为偏，异性为正）
- 阴阳属性影响大运排列方向（阳年男命/阴年女命顺排，反之逆排）

## 1.2 五行

五行是命理推断的核心逻辑引擎。

### 1.2.1 五行基本属性

| 五行 | 方位 | 季节 | 颜色 | 数字 | 脏腑 | 性质 |
|------|------|------|------|------|------|------|
| 木 | 东 | 春 | 青/绿 | 3、8 | 肝胆 | 生发、条达、曲直 |
| 火 | 南 | 夏 | 红/紫 | 2、7 | 心/小肠 | 炎上、温热、光明 |
| 土 | 中 | 四季末 | 黄/棕 | 5、0 | 脾胃 | 承载、包容、稳重 |
| 金 | 西 | 秋 | 白/金 | 4、9 | 肺/大肠 | 收敛、肃杀、刚锐 |
| 水 | 北 | 冬 | 黑/蓝 | 1、6 | 肾/膀胱 | 润下、智慧、流动 |

### 1.2.2 五行生克关系

**相生**（母子关系，生我者为母，我生者为子）：
```
木 → 火 → 土 → 金 → 水 → 木（循环）
木生火：木燃则火旺
火生土：火焚则成灰（土）
土生金：土中藏金矿
金生水：金冷则凝露（水）
水生木：水润则木长
```

**相克**（制约关系，克我者为官杀，我克者为财）：
```
木 → 土 → 水 → 火 → 金 → 木（循环）
木克土：木根破土
土克水：土堤挡水
水克火：水灭火焰
火克金：火炼金熔
金克木：金斧伐木
```

**反克**（五行力量悬殊时，被克者反而克制克者）：
```
金赖土生，土多金埋    火赖木生，木多火塞
水赖金生，金多水浊    木赖水生，水多木漂
土赖火生，火多土焦

金能克木，木坚金缺    火能克金，金多火熄
水能克火，火多水干    木能克土，土重木折
土能克水，水多土流
```

### 1.2.3 五行旺相休囚死

五行在不同月份（季节）有不同的力量状态：

| 月份 | 旺 | 相 | 休 | 囚 | 死 |
|------|----|----|----|----|-----|
| 寅卯月（春/木月） | 木 | 火 | 水 | 金 | 土 |
| 巳午月（夏/火月） | 火 | 土 | 木 | 水 | 金 |
| 辰戌丑未月（土月） | 土 | 金 | 火 | 木 | 水 |
| 申酉月（秋/金月） | 金 | 水 | 土 | 火 | 木 |
| 亥子月（冬/水月） | 水 | 木 | 金 | 土 | 火 |

- **旺**：当令之神，力量最强
- **相**：旺所生者，力量次强
- **休**：生旺者，力量平和（已休息）
- **囚**：克旺者，力量被制约
- **死**：被旺所克者，力量最弱

**在分析中的应用**：这是判断日主旺衰的第一步——"得令"还是"失令"。

# 第二章：排盘方法

## 2.1 年柱排法

**规则**：以**立春**为年份分界（不是农历正月初一）。

- 立春之前出生：属上一年
- 立春之后出生：属本年

**算法**：年份对应的天干地支可通过公式计算：
```
年天干 = (年份 - 4) % 10 → 对应索引：0甲 1乙 2丙 3丁 4戊 5己 6庚 7辛 8壬 9癸
年地支 = (年份 - 4) % 12 → 对应索引：0子 1丑 2寅 3卯 4辰 5巳 6午 7未 8申 9酉 10戌 11亥
```

**示例**：1990年
- 天干：(1990-4) % 10 = 6 → 庚
- 地支：(1990-4) % 12 = 6 → 午
- 年柱：庚午

## 2.5 大运排法

### 2.5.1 确定大运顺逆

| 条件 | 排列方向 |
|------|---------|
| 阳年男命（年干为甲丙戊庚壬） | **顺排** |
| 阴年女命（年干为乙丁己辛癸） | **顺排** |
| 阴年男命（年干为乙丁己辛癸） | **逆排** |
| 阳年女命（年干为甲丙戊庚壬） | **逆排** |

### 2.5.2 计算起运年龄

**顺排**：从出生日起，数到下一个**节**（非中气）的天数
**逆排**：从出生日起，数到上一个**节**的天数

```
起运年龄 = 天数 ÷ 3（四舍五入取整）
```

每 3 天折合 1 年，1 天折合 4 个月，1 个时辰折合 10 天。

### 2.5.3 排列大运干支

- **顺排**：从月柱的下一个干支开始，依次排列
- **逆排**：从月柱的上一个干支开始，依次排列

每步大运管 10 年（前 5 年偏重天干，后 5 年偏重地支）。

**示例**（庚午年甲申月，男命，顺排）：
```
月柱：甲申
大运：乙酉 → 丙戌 → 丁亥 → 戊子 → 己丑 → 庚寅 → 辛卯 → 壬辰
```

## 2.6 流年

流年即当年的天干地支，直接由年份推算（同年柱算法）。

大运配合流年是论命的关键——大运是"大环境"，流年是"具体事件的触发器"。

# 第三章：分析方法论

## 3.1 日主旺衰判断

日主旺衰是八字分析的基石，决定了用神的选取方向。

### 3.1.1 四步判断法

```
第一步：得令/失令（月令对日主五行的旺衰影响）
    ↓
第二步：得地/失地（地支藏干中是否有日主同类或生扶之物）
    ↓
第三步：得生/受克（天干中是否有生扶或克泄日主的力量）
    ↓
第四步：综合判断
```

#### 第一步：得令/失令

查看日主五行在月令中的旺相休囚死状态（参见 1.2.3 节）：
- **旺、相**：得令（月令生扶日主）→ 加分
- **休**：不得不失 → 中性
- **囚、死**：失令（月令克泄日主）→ 减分

**月令是旺衰判断中权重最大的因素**，约占总判断的 40%。

#### 第二步：得地/失地

检查四柱地支的藏干中，有多少是日主的同类（比劫）或生扶（印星）：

| 情况 | 判断 |
|------|------|
| 日支藏干有日主本气 | 强力得地（坐下有根） |
| 其他地支藏干有日主同类 | 得地（有根） |
| 地支藏干均无日主同类或印星 | 失地（无根） |

#### 第三步：得生/受克

检查天干中的其他三干与日主的关系：
- 有印星（生我者）→ 得生
- 有比劫（同我者）→ 得助
- 有食伤（我生者）→ 泄身
- 有财星（我克者）→ 耗身
- 有官杀（克我者）→ 克身

#### 第四步：综合判断

| 得令 | 得地 | 得生 | 综合判断 |
|------|------|------|---------|
| ✅ | ✅ | ✅ | 日主极旺 |
| ✅ | ✅ | ❌ | 日主偏旺 |
| ✅ | ❌ | ✅ | 日主中和偏旺 |
| ❌ | ✅ | ✅ | 日主中和偏旺 |
| ❌ | ✅ | ❌ | 日主中和偏弱 |
| ❌ | ❌ | ✅ | 日主偏弱 |
| ❌ | ❌ | ❌ | 日主极弱 |

**注意**：以上为简化模型。实际判断中需要考虑合化、冲刑等因素对力量的影响。

## 3.2 用神取用

用神是使八字趋于平衡的关键五行，是命理分析中最重要的判断。

### 3.2.1 扶抑取用法（最基本）

| 日主状态 | 用神方向 | 具体取用 |
|---------|---------|---------|
| 日主旺 | 抑（克、泄、耗） | 官杀克身、食伤泄身、财星耗身 |
| 日主弱 | 扶（生、助） | 印星生身、比劫助身 |

**优先级**：
- 日主旺，优先取食伤泄秀（温和），次取财星耗身，最后取官杀克身（最烈）
- 日主弱，优先取印星生身（兼有护身之功），次取比劫助身

### 3.2.2 调候取用法

某些月份出生的日主，需要"调候"（调节寒暖燥湿）：

| 日主 | 月份 | 调候用神 | 原因 |
|------|------|---------|------|
| 甲乙木 | 冬月（亥子丑） | 丙火（太阳） | 寒木需暖，无火则木不能生发 |
| 甲乙木 | 夏月（巳午未） | 癸水 | 暑木需润，无水则木枯焦 |
| 丙丁火 | 冬月（亥子丑） | 甲木 | 寒火需木生，否则火灭 |
| 庚辛金 | 夏月（巳午未） | 壬癸水 | 热金需淬，否则金熔 |
| 壬癸水 | 夏月（巳午未） | 庚辛金 | 热水需金生，否则水干 |
| 壬癸水 | 冬月（亥子丑） | 丙火 | 寒水需暖，否则冰冻 |

**调候用神优先级高于扶抑用神**——有些八字即使日主不弱，但如果缺少调候用神，也是贫贱之命。

### 3.2.3 通关取用法

当八字中两种五行力量对峙（如金木交战），取能化解两者矛盾的五行为用神：

```
金木交战 → 取水通关（金生水、水生木）
木土交战 → 取火通关（木生火、火生土）
土水交战 → 取金通关（土生金、金生水）
水火交战 → 取木通关（水生木、木生火）
火金交战 → 取土通关（火生土、土生金）
```

### 3.2.4 用神、喜神、忌神、仇神、闲神

| 名称 | 定义 | 对命局的作用 |
|------|------|------------|
| **用神** | 使命局趋于平衡的关键五行 | 最需要、最有利 |
| **喜神** | 生扶用神的五行 | 辅助用神，有利 |

# 第三章（续）：进阶分析方法论

## 3.5 干支作用关系进阶

### 3.5.1 远近亲疏原则

八字四柱之间的作用力并非均等，**紧邻者作用力最强，相隔者作用力减弱**。

```
年柱 ←→ 月柱 ←→ 日柱 ←→ 时柱
 远        近        近        远
      ↑                  ↑
   可作用              可作用
         ↑─── 隔柱作用力弱 ───↑
```

**天干作用力规则**：
| 位置关系 | 作用力 | 说明 |
|---------|--------|------|
| 紧邻（如年干与月干） | 强（100%） | 可直接合、冲、克 |
| 隔一柱（如年干与日干） | 中（约50%） | 可作用但力量减半 |
| 隔两柱（年干与时干） | 弱（约25%） | 基本不直接作用 |

**地支作用力规则**：
| 位置关系 | 作用力 | 说明 |
|---------|--------|------|
| 紧邻 | 强 | 合冲刑害均有效 |
| 隔一柱 | 中 | 冲的力量尚在，合的力量减弱 |
| 隔两柱 | 弱 | 冲力很弱，合基本无效 |

**在分析中的应用**：
- 判断两干支是否真正发生作用时，必须考虑远近
- 日柱（日干+日支）是命局核心，与日柱紧邻的月柱和时柱影响最大
- 年柱对日柱的直接作用力较弱，但年柱通过影响月柱间接影响日柱

### 3.5.2 争合与妒合

**争合**：一干对两干同时相合。

```
示例：甲 己 甲
     两个甲争合一个己 → 合的力量分散，甲己合的力量减弱
```

**妒合**：一干被两干同时合住。

```
示例：己 甲 己
     两个己争着合甲 → 甲被两己争合，犹豫不决之象
```

**规则总结**：
| 情况 | 结果 |
|------|------|
| 一合（正常合） | 两干合住，力量减弱或化出新五行 |
| 争合（一对多） | 合力分散，任何一方都合不住 |
| 妒合（多对一） | 被合方受困，但合不真切 |

### 3.5.3 合化条件详述

天干合化需要满足严格条件，**合而不化是常态**，真正合化是特例：

**合化必须满足的条件**：
1. 两干紧邻（隔柱不化）
2. 化出的五行在月令得令（如甲己合化土，需在辰戌丑未月或火土旺月）
3. 化神（化出的五行）在局中有根（地支有同类五行支撑）
4. 无克破（化出的五行不被其他强力五行克制）
5. 日干参与合化时，日干本性改变——此为"化格"的条件

**合化后的处理**：
- 合化成功：两干不再以原来的五行论，按化出的五行论
- 合而不化：两干被"绊住"，各自力量减弱约30-50%，但五行不变

### 3.5.4 地支力量层级对比

不同地支关系的力量大小排序（从强到弱）：

```
三会局 > 三合局 > 六合 > 六冲 > 三刑 > 六害（穿）
```

**力量冲突时的优先级规则**：
| 冲突情况 | 结果 |
|---------|------|
| 合与冲同时存在 | 合先于冲（"合能解冲"） |
| 三合局与六冲 | 三合成局则冲力减弱 |
| 六合与三合同时存在 | 以力量大者为主 |
| 刑与合同时存在 | 合能解刑 |

### 3.5.5 地支暗合

除明合（六合、三合）外，地支藏干之间也存在暗合关系：

| 暗合 | 藏干关系 | 含义 |
|------|---------|------|
| 寅丑暗合 | 寅中甲与丑中己（甲己合） | 暗中的情感、私下的联系 |
| 卯申暗合 | 卯中乙与申中庚（乙庚合） | 暗中的合作、私密关系 |
| 午亥暗合 | 午中丁与亥中壬（丁壬合） | 暗中的吸引、隐秘感情 |
| 巳酉暗合 | 巳中丙与酉中辛（丙辛合）（存争议）| 暗中的默契 |

**在分析中的应用**：暗合主隐秘之事，在婚姻论断中尤其重要——暗合多主暗恋、私情、地下情。

## 3.6 宫位理论

### 3.6.1 四柱宫位含义

四柱不仅代表天干地支的五行属性，还各自对应不同的人生领域和时间阶段：

| 柱位 | 别名 | 人生阶段 | 代表领域 | 年龄段 |
|------|------|---------|---------|--------|
| 年柱 | 祖上宫/父母宫 | 幼年 | 祖上、父母、出身、家族背景、社会环境 | 1-16岁 |
| 月柱 | 兄弟宫/事业宫 | 青年 | 兄弟姐妹、工作事业、社会关系、同辈 | 17-32岁 |
| 日柱 | 夫妻宫/命宫 | 中年 | 自身（日干）、配偶（日支）、家庭 | 33-48岁 |
| 时柱 | 子女宫/归宿宫 | 晚年 | 子女、晚辈、下属、事业成果、晚年生活 | 49岁以后 |

### 3.6.2 宫位与六亲对应

**六亲看法的两个维度**：
1. **十神定人**：十神决定"是谁"（如正财为妻、正官为夫）
2. **宫位定位**：宫位决定"在哪里"（如日支为夫妻宫）

| 六亲 | 十神（男命） | 十神（女命） | 所在宫位 |
|------|-----------|-----------|---------|

## 3.8 大运流年作用法则

### 3.8.1 大运与原局的作用路径

大运对原局的作用遵循层级规则：

```
大运天干 → 作用于原局天干（直接作用）
大运地支 → 作用于原局地支（直接作用）
大运天干 ✗ 原局地支（天干不直接作用地支）
大运地支 ✗ 原局天干（地支不直接作用天干）
```

**大运天干地支的时间分配**：
| 时段 | 主导 | 说明 |
|------|------|------|
| 大运前5年 | 天干为主 | 天干的影响占主导 |
| 大运后5年 | 地支为主 | 地支的影响占主导 |
| 全程 | 干支同时作用 | 只是侧重不同，非一方完全不作用 |

### 3.8.2 流年引发机制

流年是具体事件的"触发器"。大运决定了十年的大环境，流年决定具体哪一年应验。

**引发条件**（以下任一条件满足即可触发事件）：

| 引发方式 | 说明 | 示例 |
|---------|------|------|
| 冲 | 流年地支冲原局或大运的关键地支 | 流年子冲原局午 → 午所代表的事项发生变动 |
| 合 | 流年合住原局的某柱 | 流年丑合原局子 → 子所代表的事项被合动 |
| 刑 | 流年与原局/大运形成三刑 | 流年巳+原局寅+大运申 → 无恩之刑触发 |
| 值 | 流年干支与原局某柱相同 | 流年甲午，原局有甲午 → 该柱信息被激活 |
| 透 | 流年天干透出原局藏干 | 流年庚年，原局申中庚透出 → 庚的十神含义激活 |

### 3.8.3 应期判断（精细化时间）

```
大运 → 确定十年运势方向
流年 → 确定具体哪一年
流月 → 确定具体哪个月（可选，高级功能）
```

**应期判断规则**：
| 事件类型 | 应期看法 |
|---------|---------|
| 结婚 | 男看流年遇财星（正财/偏财），女看流年遇官星（正官） |
| 升职 | 流年遇正官/正印，且官印得用 |
| 破财 | 流年遇劫财/比肩克财，或财星入库/被冲 |
| 搬家 | 流年冲动驿马，或冲日支/年支 |
| 生子 | 男看流年官杀（子女星），女看流年食伤（子女星） |
| 健康 | 流年冲克用神对应五行 → 用神五行对应脏腑出问题 |

### 3.8.4 大运转折点

**交运年**（大运切换的年份）是人生重要的转折年，需要特别关注：

- 交运年前后1-2年容易有大事发生
- 如果新旧大运五行属性差异大（如从木运转金运），转折感更明显
- 交运年如果又遇到流年冲合原局，事件感最强

## 3.10 特殊命局详论

### 3.10.1 从格详论

从格是日主极弱，无法自立，弃命从势的格局。

**从格的严格条件**：
1. 日主在月令处于死绝之地
2. 四柱天干无印星、无比劫来帮扶日主
3. 地支无日主的根气（藏干无同类五行）
4. 日主被财官食伤团团包围

**各类从格**：

| 从格类型 | 条件 | 用神 | 忌神 |
|---------|------|------|------|

| 从势格 | 财官食伤混合极旺 | 顺从旺势 | 印星、比劫 |

**从格论命特点**：
- 从格之人往往在某一领域有极端的表现（要么大富大贵，要么穷困潦倒）
- 从格最怕行印比运（破格），一旦走到印比运，就是大凶之运
- 真从格是富贵命，假从格（似从非从）反而是贫贱命

## 3.11 刑冲合害进阶应用

### 3.11.1 合中有克

六合虽然主合好，但合的五行之间也存在生克关系：

| 六合 | 合中关系 | 吉凶倾向 |
|------|---------|---------|
| 子丑合 | 子水被丑土克 → 水受损 | 合中带克，有得有失 |
| 寅亥合 | 亥水生寅木 → 合中带生 | 纯合，最为吉利 |
| 卯戌合 | 戌中辛金克卯木 → 合中带克 | 合好但有隐患 |
| 辰酉合 | 酉金生辰中癸水 → 但辰为酉墓 | 合中复杂 |
| 巳申合 | 申金克巳中甲木 → 合中有克 | 合中带争 |
| 午未合 | 午火生未土 → 合中带生 | 较吉利 |

### 3.11.2 冲的进阶判断

并非所有冲都是坏事，要看**谁冲谁、冲的是用神还是忌神**：

| 情况 | 结果 |
|------|------|
| 冲去忌神 | 大吉（去除不利因素） |
| 冲去用神 | 大凶（失去有利因素） |
| 冲动财库（辰戌冲、丑未冲） | 可能发财（库被冲开，财物流出） |
| 冲动驿马 | 出行、搬迁、工作变动 |
| 日月冲 | 工作或家庭动荡 |
| 日时冲 | 与子女关系不和，或晚年不安 |
| 年月冲 | 离祖别家，早年环境变动大 |

### 3.11.3 墓库刑冲详论

| 库 | 藏什么 | 含义 |

| 辰 | 水库 | 藏壬癸水、乙木、戊土 |
| 戌 | 火库 | 藏丁火、辛金、戊土 |
| 丑 | 金库 | 藏辛金、癸水、己土 |
| 未 | 木库 | 藏乙木、丁火、己土 |

**库的状态**：
- **入库**：某五行的天干无力且地支坐库 → 被关在库中，力量无法发挥
- **库开**：库被冲开（辰戌冲、丑未冲）或被刑开（丑戌未三刑）→ 库中五行释放
- **墓**：日主入库（如丙火日主见戌）→ 日主入墓，力量极弱

**库开与财运的关系**：
```
财星入库 + 库被冲开 → 得财之象（"财库冲开则发财"）
财星入库 + 库未被冲 → 有财但难以到手（"财在库中取不出"）
```

## 3.12 断事实战技法总结

### 3.12.1 断事的核心公式

```
事件 = 星（十神代表什么人/事）
     × 宫（在哪个柱位发生）
     × 象（具体是什么样的人/事/物）
     × 时（大运流年何时触发）
     × 力（旺衰决定结果好坏程度）
```

### 3.12.2 常见事件的断法速查

| 事件 | 看什么星 | 看什么宫 | 触发条件 |
|------|---------|---------|---------|
| 升职 | 正官/正印 | 月柱（事业宫） | 流年遇官印且得用 |
| 失业 | 偏官/伤官 | 月柱（事业宫） | 伤官见官，或官星被冲克 |
| 结婚 | 正财（男）/正官（女） | 日支（夫妻宫） | 流年合动日支，或流年遇配偶星 |
| 离婚 | 劫财克财/伤官克官 | 日支被冲 | 流年冲日支，且忌神当道 |
| 生育 | 食伤（女）/官杀（男） | 时柱（子女宫） | 流年遇子女星且时柱被合动 |
| 发财 | 正财/偏财 | 不限 | 流年行财运，且食伤生财 |
| 破财 | 劫财/比肩 | 不限 | 流年遇比劫夺财 |
| 搬家 | 驿马/冲 | 年柱或日柱 | 驿马逢冲，或年/日支被冲 |
| 考试 | 正印/食神 | 月柱 | 流年遇印星或文昌 |
| 手术 | 偏官/羊刃 | 对应身体宫位 | 流年遇官杀克身+羊刃 |
| 车祸 | 偏官+驿马 | 日柱/时柱 | 流年冲驿马且官杀旺 |

### 3.12.3 吉凶层次判断

不是所有吉都是大吉，不是所有凶都是大凶。层次取决于力量：

| 层次 | 条件 |
|------|------|
| 大吉 | 用神得力+格局纯正+大运流年同助用神 |
| 小吉 | 用神得力但有小瑕疵，或助力不够强 |
| 平 | 用神与忌神力量均衡 |
| 小凶 | 忌神稍旺，或用神稍受克 |
| 大凶 | 用神被彻底克破+格局被破+大运流年同攻用神 |

### 3.12.4 分析时的常见误区

| 误区 | 正确做法 |
|------|---------|
| 五行缺什么就补什么 | 应该看用神是什么才补什么，缺不等于需要 |
| 只看天干不看地支 | 地支是根基，力量更大更稳定 |
| 只看旺衰不看格局 | 格局决定命局层次和方向 |
| 忽视调候 | 调候用神有时比扶抑更重要 |
| 神煞决定一切 | 神煞是辅助参考，不能替代十神分析 |
| 把大运当流年看 | 大运是大环境，流年是触发器，层级不同 |
| 看到冲就说凶 | 冲去忌神反而是好事 |
| 纳音定命 | 纳音仅做参考和意象，不做核心判断 |

# 第四章：论命流程

## 4.1 完整论命十步法

```
第一步：排盘
    准确排出四柱八字（年柱、月柱、日柱、时柱）
    ↓
第二步：标注信息
    标注藏干、纳音、十神、神煞
    ↓
第三步：判断日主旺衰
    按四步判断法（得令→得地→得生→综合）
    ↓
第四步：确定用神
    扶抑法为主，兼顾调候和通关
    ↓
第五步：判定格局
    月令取格 → 成格/破格判断 → 格局层次
    ↓
第六步：分析十神配置
    看十神的分布、强弱、组合关系
    ↓
第七步：分析地支关系
    冲、合、刑、害的影响
    ↓
第八步：排大运、看流年
    每步大运与原局的作用 → 具体年份的吉凶
    ↓
第九步：综合论断
    性格、事业、财运、婚姻、健康等各方面
    ↓
第十步：给出建议
    基于用神五行的实用建议
```

## 4.2 各领域论断方法

### 4.2.1 事业论断

| 看什么 | 怎么看 | 结论 |
|--------|--------|------|
| 官杀 | 官杀为"工作、事业、领导" | 官杀旺且为用：仕途通达 |
| 食伤 | 食伤为"才华、技艺、表达" | 食伤旺且为用：技术/创意类 |
| 财星 | 财星为"钱、资源、商业" | 偏财旺：善于经商投资 |
| 印星 | 印星为"学历、权力背书" | 印旺：适合学术/教育 |
| 驿马 | 驿马星主变动、出差 | 驿马逢冲：工作变动多 |

### 4.2.2 财运论断

| 看什么 | 怎么看 | 结论 |
|--------|--------|------|
| 正财 | 正财为稳定收入 | 正财旺且为用：工薪收入好 |
| 偏财 | 偏财为意外之财 | 偏财旺且为用：投资理财有回报 |
| 比劫 | 比劫为争财之神 | 比劫旺而财弱：破财之象 |
| 食伤生财 | 才华转化为钱 | 有此组合：靠本事赚钱 |
| 大运财运 | 行财运时 | 大运走财地：财运旺盛 |

### 4.2.3 婚姻论断

**男命看财星**（正财为妻，偏财为情人）：
- 正财旺且纯：婚姻稳定
- 偏财旺或财多：感情复杂
- 比劫克财：婚姻有压力

**女命看官杀**（正官为夫，偏官为情人）：
- 正官旺且纯：婚姻稳定
- 官杀混杂：感情复杂
- 伤官克官：婚姻有波折

**婚期看法**：
- 男命：大运/流年遇财星（正财）
- 女命：大运/流年遇官星（正官）
- 合多：有合之年容易结婚

### 4.2.4 健康论断

| 五行 | 对应脏腑 | 过旺的问题 | 过弱的问题 |
|------|---------|-----------|-----------|
| 木 | 肝胆、眼睛、筋骨 | 肝火旺、头痛 | 肝功能弱、视力问题 |
| 火 | 心脏、小肠、血液 | 心火旺、失眠 | 心脏功能弱、血压低 |
| 土 | 脾胃、肌肉、口腔 | 脾胃湿热 | 消化不良、营养吸收差 |
| 金 | 肺、大肠、皮肤、呼吸 | 肺燥、皮肤干 | 呼吸系统弱、易感冒 |
| 水 | 肾、膀胱、泌尿、骨骼 | 肾水过旺、水肿 | 肾虚、骨质问题 |

**健康问题的触发**：大运/流年冲克用神五行对应的脏腑时，最易出现健康问题。

### 4.2.5 性格论断

性格主要由以下因素综合决定：

1. **日干本性**（参见 1.3.1 十天干性质描述）
2. **十神格局**（格局决定性格基调）：
   - 正官格：正直规矩
   - 偏官格：果断霸气
   - 正印格：仁慈好学
   - 偏印格：多思求异
   - 正财格：勤俭务实
   - 偏财格：慷慨大方
   - 食神格：温和享受
   - 伤官格：聪明叛逆
3. **日主旺衰**（旺者自信主动，弱者谨慎被动）
4. **五行偏重**（哪种五行最强就带有该五行性格特质）

4. 允许命理师在审核时切换方案
