# 命理报告输出方案

> **文档性质**：生产 SOP —— 从"拿到命主八字"到"交付三文件"的完整工作流程与质量标准
> **模板来源**：`蔡命完整分析-v4-命理师版.md` / `蔡命完整分析-v4-消费者版.md` / `蔡命完整分析-v4-索引.md`
> **设计决策**（已拍板）：
> - 使用场景：人工喂八字 → AI 稳定产出 → 后续对接排盘引擎
> - 流年窗口：以大运交脱为锚点（当前大运剩余年 + 下步大运前 3 年），流月固定 12 个月
> - 典型画像：完全动态（每个命主重新生成）
> - 命理师版深度：保持 900+ 行完整推演
> - 信息深度分两档：纯八字通用版 / 采集基础信息后适配版

## 0. 核心输出哲学（最高优先级·所有模块必须遵守）

> **一句话定义**：命理报告 = 用**命理学的精度** + **人话的温度** + **战略顾问的高度**，产出一份让命主看完就知道"我是谁、我在哪、我接下来该干什么、该避开什么"的**人生操作手册**。

### 0.1 三条铁律

| # | 铁律 | 说明 | 正例 | 反例 |
|---|---|---|---|---|
| **P1** | **命理是骨架，人话是皮肉** | 每条命理术语后**必须紧跟白话翻译**——不是文末注释，是**原地翻译、当场消化**。命理师版和消费者版均适用。 | `印比杀运 → 典型的"读了很多书但没赚到多少钱"的前半生` | `印比杀运 → 身旺有印、学业强、但财运不显` |
| **P2** | **有战略意图，不是零散信息堆叠** | 报告必须用一个**人话级别的"战略意图"**把所有结论串成一条线。命主看完要能记住"我的人生主线是什么"，而不是"我记了一堆术语"。 | `未来2年所有事情的本质都是"为47岁那波财运转折储备弹药"——拿头衔、攒人脉、管好身体` | 大运结论罗列5条但没有总结"所以呢" |
| **P3** | **专业性体现在精准定位，不是术语密度** | 真正的专业 = 像资深命理师坐在对面跟你说话——他知道所有术语含义，但选择用你听得懂的方式告诉你"这对你意味着什么"。术语是**证据链**（让你知道我不是瞎编的），人话才是**结论**。 | `双戌印库 = 你天生是"考试体质"，学什么都比别人快，但问题是学了不赚钱` | `双戌印库 = 印旺无制 · 中等偏上` |

### 0.2 人话翻译的三种固定句式

所有白话翻译**必须使用以下三种句式之一**，不允许自由发挥导致风格不统一：

```
句式 A（最常用）：
  「{术语}」翻译成人话 = {日常感受/生活场景描述}

句式 B（大运/流年结论后）：
  → {一句话白话概括，不含任何术语}

句式 C（类比句）：
  就像{日常类比} → {命理含义}
```

**示例**：
```
句式 A：「伤官见官」翻译成人话 = 你的才华和权威天生打架，说话越精彩越容易得罪人
句式 B：己酉→戊申→丁未 = 印比杀运 → 典型的"读了很多书但没赚到多少钱"的前半生
句式 C：就像一个力量超标的运动员 → 力量大不是问题，找不到出口才是问题
```

### 0.3 「当下定位」段规范（命理师版 + 消费者版均需）

**每份报告必须在大运表格后、Part 1 前，插入一个 🎯 当下定位段**。这是整份报告的"战略锚点"，所有后续建议都围绕它展开。

**必选要素**（4 条缺一不可）：

| # | 要素 | 模板 | 示例 |
|---|---|---|---|
| 1 | **当前坐标** | 你正好处在第 N 步大运「XX」的{位置}（虚岁 X-X 第 N 年）——{用忌定性}还剩 N 年 | "你正好处在第4步大运「丙午」正官运的中后段——这步大运还剩2年" |
| 2 | **转折倒计时** | 距离{关键转折}还剩 N 年（{年份} {虚岁}岁正式入「XX」运） | "距离命运级转折点还剩2年（2028年47岁入「乙巳」偏财运）" |
| 3 | **战略意图** | 这意味着：未来 N 年你做的所有事情，本质都是"{一句话战略}"——{3-5个具体动作} | "本质都是为47岁那波财运转折储备弹药——拿头衔、攒人脉、考资质" |
| 4 | **当前叠加状态**（可选） | 当前叠加：{流年}+{大运}= {人话解读} | "2026流年丙午=大运丙午 → 双重正官 = 近10年存在感最强也最累的一年" |

**蔡命实例（已验证）**：
```
> 🎯 蔡命当下定位（2026-05，33 虚岁）：
> - 你正好处在第 3 步大运「庚申」的中段（虚岁 31-40 第 4 年）——忌神运 7 年还剩 7 年。
> - 距离用神大运启动还剩 7 年（2033 癸丑年 41 岁正式入「己未」运）。
> - 这意味着：未来 7 年你做的所有事情，本质都是"为 41 岁那波主升浪储备弹药"——读书、考证、攒人脉、把作品集做扎实、管好身体、谈恋爱锁定配偶。这是命书给你的"战略意图"，所有定制化建议都围绕这个意图展开。
```

## 1. 总览

### 1.1 输入 → 输出

```
输入                              输出
─────────────────                 ─────────────────
必选：命主八字排盘数据              ① {姓}命完整分析-命理师版.md（900+ 行）
可选：命主基础信息采集表            ② {姓}命完整分析-消费者版.md（1000+ 行）
                                  ③ {姓}命完整分析-索引.md（~80 行）
```

### 1.2 两档模式

| 档位 | 输入 | 消费者版差异 | 典型场景 |
|---|---|---|---|
| **A 档 · 纯八字** | 仅排盘数据 | 趋利避害用通用建议（按五行/十神规则推导） | 批量出报告 / 命主未提供更多信息 |
| **B 档 · 采集适配** | 排盘数据 + 基础信息采集表 | 趋利避害绑定命主现状（行业/城市/婚恋/健康/年龄） | 深度服务 / 高付费客户 |

**差异范围**：仅消费者版的"趋利避害建议"部分受档位影响；命理师版始终保持完整推演（命理事实不因信息深度而改变）。

### 1.3 工作流程（SOP）

```
Step 1 · 输入准备
  ├─ 必选：获取排盘数据（排盘引擎 or 手工输入）
  └─ 可选：命主填写基础信息采集表（B 档）

Step 2 · 命理师版生成（按 Part 0-4 顺序）
  ├─ Part 0：基础排盘信息落盘
  ├─ Part 1：命局核心结构分析（旺衰→用神→格局→十神）
  ├─ Part 2：干支动力关系（合冲刑害→神煞）
  ├─ Part 3：大运流年深度（大运分段→流年清单）
  └─ Part 4：六领域深度分析（婚姻/财富/事业/健康/六亲/自我）

Step 3 · 消费者版生成（基于命理师版翻译）
  ├─ 卷首：三句话总结
  ├─ 板块 ①：命书底牌（5 个基础事实）
  ├─ 板块 ②：你这个人（人格/路径/命门/红利）
  ├─ 板块 ③：你和你的人（6 张六亲卡片）
  ├─ 板块 ④：你的时间表（大运/流年/流月）
  ├─ 板块 ⑤：典型画像（完全动态生成）
  └─ 卷尾：3 张速查卡

Step 4 · 索引文件生成

Step 5 · 质量校验（三文件协同校验清单）

Step 6 · 事件咨询（按需·命主带着具体决策来问时触发）
  ├─ 四步法：定位时空坐标 → 拆解事件命理属性 → 逐层匹配 → 输出趋利避害
  ├─ 方法论 & 案例库 → `docs/事件咨询方法论与案例库.md`
  └─ 输出独立存档 → `{姓}命事件咨询-{事件关键词}-{YYYY-MM-DD}.md`
```

## 3. 命理师版输出骨架

> **总体规格**：900+ 行 / 5 个 Part / 每条结论必须标注权重 + 置信度 + 命理依据链

### Part 0 · 命主基础信息（~100 行）

| 小节 | 内容 | 写作规则 |
|---|---|---|
| §0.1 出生信息 | 公历/农历/节气/真太阳时 | 必须标注节气精确时间 + "月柱贴节气头"风险 |
| §0.2 四柱排盘 | 表格：柱位/干支/十神/纳音/藏干 | 藏干必须标本/中/余三层 |
| §0.3 五行能量分布 | 公式 + 逐项计分 + 动态调整说明 | 每条调整必须标注依据 |
| §0.4 起运与大运排列 | 起运计算 + 8 步大运全表 | 表格含：干支/干十神/起止虚岁/起止年份/用忌定性 |
| **§0.5 大运结构性结论 + 🎯当下定位** | 大运主线总结 + 命主当前坐标 + 战略意图 | ⚠️ **必选段**，详见下方规范 |

**§0.5 写作规范**：

大运结构性结论必须包含两部分：

**Part A · 大运主线一句话**（加粗·放在结论段首行）：
```
> **大运结构性结论**：**{前N年描述} + {当前运描述} + {转折运描述} = "{一句话人生主线概括}"主结构**。
> 所有后续维度结论都需以此时间轴为底色。
```
示例：`"前半生学本事、中间扛压力、后半生才赚钱"主结构` / `"先难后易、大器晚成"主结构`

**Part B · 每步大运结论必须附白话翻译**：
```
> - **{时段}**：{干支序列} = **{十神属性}** → {命理结论} → {白话翻译（句式B）}
```
❌ 反例：`印比杀运 → 身旺有印、学业强、但财运不显`（纯命理术语，命主看不懂）
✅ 正例：`印比杀运 → 典型的"读了很多书但没赚到多少钱"的前半生`（一句人话）

**Part C · 🎯当下定位段**（格式见 §0.3 核心输出哲学）

### Part 1 · 命局核心结构（~200 行）

| 小节 | 内容 | 写作规则 |
|---|---|---|
| §1.1 旺衰判定 | 得令/得地/得生/受克四步推断 | 权重 100，必须标置信度；附趋利避害 |
| §1.2 用神/忌神 | 用神推导 + 调候/扶抑/通关三法收敛 | 权重 100，必须标柱位（位置力学）；附用神时段×对象×具体动作三维趋利避害 |
| §1.3 格局判定 | 格局类型 + 成格条件 + 典型画像 | 权重 80+，画像覆盖自我/六亲/事业/财富/健康/婚姻 |
| §1.4 十神心性 | 十神分布全景 + 组合关键判读（按权重降序） | 每个组合：权重 + 外显/实质 + 位置触发条件 + 趋利避害 |

### Part 2 · 干支动力关系（~130 行）

| 小节 | 内容 | 写作规则 |
|---|---|---|
| §2.1 合冲刑害 | 一览表 + 关键关系深度解读 | 每条：权重 + 外显/实质 + "宫位被改变" + 趋利避害 |
| §2.2 神煞 | 全表（含查法依据）+ 吉/凶/中性分组判读 | 每项：来源柱位 + 查法 + 组合判读 + 趋利避害 |

### Part 3 · 大运流年深度（~200 行）

| 小节 | 内容 | 写作规则 |
|---|---|---|
| §3.1 大运分段总论 | 按"忌神段 / 交脱段 / 用神段 / 晚年段"分阶段 | 每阶段：权重 + 表格 + 趋利避害。⚠️ **每步大运结论后必须附句式B白话翻译**（见§0核心输出哲学），禁止纯术语堆叠 |
| §3.2 关键流年清单 | **以大运交脱为锚点**：当前大运剩余年 + 下步大运前 3 年 | 表格每行：公历/干支/虚岁/干十神/支与原局关系/性质/事件预测/具体怎么做 |

**流年窗口计算规则**（Q2 决策）：
```
设命主当前虚岁 = A，当前大运结束虚岁 = E
  已过流年 = max(3, A - 当前大运开始虚岁) 条回溯
  未来流年 = (E - A) + 3 条预测
  流月 = 固定 12 个月（从报告生成月起向后 12 个月）
```

### Part 4 · 六领域深度分析（~300 行）

六个领域统一结构（每域 ~50 行）：

```
### 4.X {领域名} · 评级 ⭐×N {一句话评语}

#### 🎯 该领域的核心宫位
  （标注哪一柱主管该领域 + 该柱的十神/藏干/合冲状态）

#### L1 命理事实
  （纯命理描述，不翻译，按权重降序列 3-5 条）

#### L2 对命主的影响
  （翻译成日常感受 / 生活场景，按权重降序列 3-5 条）

#### L3 🛠️ 趋利避害
  （时段 × 对象 × 具体动作 三维细分）
  A 档：通用建议（按五行/十神规则推导）
  B 档：绑定命主现状（行业/城市/婚恋/健康/年龄）
```

**六领域固定清单**：婚姻 → 财富 → 事业 → 健康 → 六亲 → 自我

## 4. 消费者版输出骨架

> **总体规格**：1000+ 行 / 卷首 + 5 板块 + 卷尾 / 全文禁术语链堆叠

### 卷首 · 三句话讲完你的命（~20 行）

```
结构（三句话固定结构）：
  第一句 · 你是谁      → 用一个意象概括命格底色
  第二句 · 你的命门     → 讲最大的风险 + "开运按钮"
  第三句 · 你的红利     → 讲主升浪时间点 + 激励
```

**写作规则**：
- 不出现任何术语（壬水/戊土/比劫等一律翻译）
- 必须包含具体年龄/年份（"41 岁" "2033 年"）
- 第三句必须以"现在很苦不要紧"类共情句收尾

### 板块 ① · 命书底牌 · 6 个基础事实（~180 行）

| 小节 | 来源 | 消费者版写法 |
|---|---|---|
| 1.1 四柱排盘 | 命理师版 §0.2 | 表格保留 + 加"🔴 这张表告诉你什么"翻译段 |
| 1.2 五行能量分布 | 命理师版 §0.3 | 柱状图保留 + 加"🔴 这张图告诉你什么"翻译段（含吃/穿/住落地建议）|
| 1.3 旺衰判定 | 命理师版 §1.1 | 只保留结论 + "🔴 这个结论告诉你什么"翻译段（含"同八字 X% 翻车"数据 + 毕生第一要务 + 千万别做清单）|

## 6. 写作铁律

### 6.1 命理师版铁律

1. **每条结论必须标注**：权重（0-100）+ 置信度（⭐1-5）+ 命理依据链
2. **推演过程不可省略**：旺衰四步推断 / 用神三法收敛 / 格局成败检查 必须完整展示
3. **趋利避害三维细分**：时段 × 对象 × 具体动作（不允许"注意婚姻"这种笼统建议）
4. **位置力学嵌入**：每个十神/神煞/合冲必须标注"在哪一柱 → 影响哪个宫位 → 什么时候触发"
5. **B 档适配部分明确标注**：用 `（绑定命主现状：{行业}/{城市}/{婚恋}/{年龄}）` 标注
6. **⚠️ 人话翻译铁律（P1·最高优先级）**：命理师版虽然面向专业读者，但**所有结论性段落必须附白话翻译**（使用§0定义的句式A/B/C之一）。具体要求：
   - **大运结构性结论**：每步大运后必须有 `→ {一句白话}`，不允许以纯术语结尾
   - **十神组合判读**：每个组合的"命理判读"段末尾必须有"翻译成人话"句
   - **六领域 L2 层**：L2 已经是翻译层，但 L1 的关键结论也需在行内附 `→ {白话}` 注释
   - **🎯当下定位段**：必选，格式见§0.3
   - **自检标准**：命理师版全文，任意连续 5 行不得出现"全是术语、没有一句人话"的情况

### 6.2 消费者版铁律

1. **三段式**：每条"🔴 核心影响"必须包含"术语 → 生活场景翻译 → 典型后果举例"
2. **翻译公式**：`「{术语}」翻译成人话 = {日常感受描述}` 或 `「{术语}」翻译成日常感受 = {描述}`
3. **狼度标准**：全文保持"命书级判决"强度——敢说"同八字 80% 都..."、敢用"毕生第一要务""最大悲剧""死命令"
4. **禁术语链堆叠**：禁止出现"七杀虚透伤官见官印星受制"这种连续 3 个以上术语无翻译的句子
5. **具体年龄/年份必现**：每个核心影响必须包含至少 1 个具体年龄或年份
6. **归因于命格**：所有负面描述归因于"命定的"而非"你的缺陷"（"这不是你不够努力，是命书..."）
7. **B 档适配**：趋利避害部分如有命主现状信息，绑定到行业/城市/婚恋/健康具体场景
8. **⚠️ 建议可执行性铁律（P3）**：趋利避害不允许停留在"多表达、多输出"这种抽象层面，必须精确到**可执行动作**（"每周游泳2-3次" / "11点前入睡" / "说话前默数5秒"），命主看完就能**直接做**

### 6.3 两版一致性铁律

1. **结论不矛盾**：消费者版与命理师版若冲突，以命理师版为准
2. **关键命门覆盖**：命理师版的所有权重 ≥ 70 的结论，消费者版必须有对应翻译
3. **时间节点一致**：大运/流年/流月的年份、干支、虚岁在两版中必须完全一致

## 8. 命理专业性保障

> **核心原则**：严禁无中生有。所有结论必须能从排盘数据 → 命理规则 → 推演过程三步追溯。

### 8.1 命理推演的依据链要求

每一条命理结论必须满足：
```
排盘数据（四柱/藏干/十神/神煞）
  ↓ 引用具体的命理规则
命理规则（方法论文档 or 典籍出处）
  ↓ 展示推演过程
推演结论（旺衰/用神/格局/十神组合判读/领域判断）
  ↓ 翻译为影响
对命主的影响（生活场景描述）
  ↓ 给出具体建议
趋利避害（时段×对象×动作）
```

### 8.2 禁止事项

1. **禁止编造命理规则**：所有规则必须有典籍或方法论文档出处
2. **禁止过度解读**：单一信号不构成结论，必须有"原局 + 大运/流年"双重验证
3. **禁止通用化**：每条结论必须能精确对应到命主八字的具体干支/宫位/十神
4. **禁止忽略反面**：看到吉象必须检查是否有凶象制约，反之亦然
5. **禁止断章取义**：引用术语（如"伤官见官"）必须检查是否满足完整触发条件
